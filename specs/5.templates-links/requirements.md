# templates-links — 需求规格

## 概述

算法模板库（按类型分类的代码片段管理）+ 题目关联（双向多对多，将相似题目关联在一起便于组合复习）。

## 项目信息

- 项目名: algorithm-tracker
- 架构类型: Turborepo monorepo

## 需求版本

| 日期       | 版本 | 说明     |
| ---------- | ---- | -------- |
| 2026-06-17 | v1   | 初始需求 |

## 用户故事

- 作为用户，我想要在模板库按类型查看代码片段，以便写题时快速复用。
- 作为用户，我想要新增/编辑/删除算法模板，以便维护自己的模板集合。
- 作为用户，我想要在题目详情页关联相似题目，以便组合复习同类型题目。
- 作为用户，我想要在题目详情页看到已关联的相似题目并跳转，以便一次巩固同类型。

## 功能需求

1. [F-001] 模板字段：类型（文本，如"动态规划"、"二分"）、代码片段、适用场景说明（可选）
2. [F-002] 模板列表页（/templates）：按类型分组展示，支持关键词搜索（匹配类型或适用场景）
3. [F-003] 支持新增、编辑、删除模板（内联模态框或独立表单页，代码区高亮展示）
4. [F-004] 题目详情页提供"关联题目"搜索框，从现有题目中搜索并添加关联
5. [F-005] 关联为双向：A 关联 B，B 的详情页也自动显示 A
6. [F-006] 可从详情页移除某条关联
7. [F-007] 关联列表显示：被关联题目的标题、难度、标签

## 非功能需求

- 安全: 所有操作使用 `protectedProcedure`

## 验收标准

- [ ] [AC-001] 新增模板后，/templates 页对应类型下出现该模板
- [ ] [AC-002] 搜索"二分"能匹配到 type="二分" 的模板
- [ ] [AC-003] 题目A关联题目B后，B的详情页也显示A在关联列表中
- [ ] [AC-004] 从A移除对B的关联后，B的详情页也不再显示A
