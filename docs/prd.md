# PRD — 算法学习追踪器

Version: 1.0
Date: 2026-06-17

## 1. 产品概述

个人使用的算法刷题追踪工具，记录做题进度、错题复盘、模板复用，并通过 /workflow 页展示AI开发流程的Mermaid可视化。

## 2. 用户故事

| ID | As a user, I want to... | So that... |
|----|--------------------------|------------|
| US-01 | 用固定账号登录 | 不需要注册流程也能进入个人数据 |
| US-02 | 在看板首页看到统计图表和连续打卡天数 | 直观了解自己的刷题节奏 |
| US-03 | 在题目记录页增删改查题目记录，按标签/难度筛选 | 方便管理做过的题 |
| US-04 | 在错题本标记某题"需要二刷" | 集中复习薄弱点 |
| US-05 | 收到基于艾宾浩斯曲线(1/3/7/15/30天)的复习提醒 | 不会忘记该复习的错题 |
| US-06 | 在算法模板库按类型查看代码片段 | 写题时能快速复用模板 |
| US-07 | 把相似题目关联到一起组合复习 | 一次性巩固同类型题目 |
| US-08 | 在 /workflow 页看到Mermaid可视化的AI开发流程图 | 满足课程作业要求 |

## 3. 功能需求细节

### 3.1 看板首页
- 展示总刷题数、本周刷题数、连续打卡天数
- 图表展示近30天做题趋势（按天聚合）
- 连续打卡：当日有新增题目记录则计入连续天数，中断则清零

### 3.2 题目记录
- 字段：题目名称、来源（LeetCode/牛客等）、难度（简单/中等/困难）、标签（数组）、做题日期、用时、是否一次AC、笔记
- 支持新增/编辑/删除/查看详情
- 支持按标签、难度、是否AC筛选

### 3.3 错题本
- 从题目记录中标记"需要二刷"的题目自动进入错题本
- 复习提醒按艾宾浩斯曲线计算：标记当天起第1/3/7/15/30天提醒复习一次
- 完成一次复习后，"已复习次数+1"，下一次提醒按曲线顺延
- 5次复习全部完成后移出"待复习"列表，归档

### 3.4 算法模板库
- 模板字段：类型（如动态规划/二分/双指针）、代码片段、适用场景说明
- 按类型分类展示，支持搜索

### 3.5 题目关联
- 题目记录可手动关联到其他题目（多对多）
- 题目详情页展示关联的相似题目列表，点击可跳转

### 3.6 /workflow 页
- 用Mermaid展示本项目AI辅助开发的工作流程图（init→prd→ai三阶段）
- 纯展示性质，不需要交互逻辑

## 4. 非功能需求

| 类别 | 要求 |
|------|------|
| 性能 | 列表页加载 < 1s（个人项目，数据量小） |
| 安全 | 固定账号登录，无需多用户权限隔离；密钥走环境变量，不硬编码 |
| 成本 | 优先使用 AWS Always Free 额度内服务 |

## 5. Out of Scope（本次不做）

- Jira Bug列表自动拉取 + EventBridge定时触发 + opencli自动修复（课程作业明确为"后期"功能）
- 多用户注册/权限系统
- 移动端适配（桌面端优先）
- 题目内容全文搜索
